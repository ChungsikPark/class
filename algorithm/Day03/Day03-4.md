- ### Question

  **문제 설명**

  num을 입력받아 1부터 num까지의 숫자 중 홀수로 구성된 문자열을 만들어야 합니다.
  num에 5일 경우에는 "135"입니다.

  **제한 조건**

  - num은 숫자열입니다.

  입출력 예

  ```jsx
  Input: makeOdd(5);
  Output: 135;

  Input: makeOdd(7);
  Output: 1357;
  ```

---

- ### My Code

  - Solution

    ```jsx
    const makeOdd = (num) => {
      let result = "";
      for (let i = 1; i <= num; i++) {
        if (i % 2) {
          result += i;
        }
      }
      return result;
    };
    ```
