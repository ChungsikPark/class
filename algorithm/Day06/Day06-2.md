- ### Question

  **문제 설명**

  단어 s의 가운데 글자를 반환하는 함수, solution을 만들어 보세요. 단어의 길이가 짝수라면 가운데 두글자를 반환하면 됩니다.

  **제한 조건**

  - s는 길이가 1 이상, 100이하인 스트링입니다.

  **입출력 예**

  ```jsx
  Input: "abcde";
  Output: "c";

  Input: "qwer";
  Output: "we";
  ```

---

- ### My Code

  - Solution

    ```jsx
    const solution = (s) => {
      return s.length % 2
        ? s[Math.floor(s.length / 2)]
        : s[s.length / 2 - 1] + s[s.length / 2];
    };
    ```
