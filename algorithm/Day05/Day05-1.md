- ### Question

  **문제 설명**

  정수 num이 짝수일 경우 "Even"을 반환하고 홀수인 경우 "Odd"를 반환하는 함수, solution을 완성해주세요.

  **제한 조건**

  - num은 int 범위의 정수입니다.
  - 0은 짝수입니다.

  **입출력 예**

  ```jsx
  Input: 3;
  Output: "Odd";

  Input: 4;
  Output: "Even";
  ```

---

- ### My Code

  - Solution

    ```jsx
    const solution = (num) => {
      return num % 2 ? "Odd" : "Even";
    };
    ```

- ### Reference Code

  - Solution

    ```jsx
    const solution = (num) => {
      let result = "";
      if (num % 2 === 0) {
        result = "Even";
      } else {
        result = "Odd";
      }
      return result;
    };
    ```
